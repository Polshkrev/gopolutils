# Среда Разработки
Модуль для стандартизации установки, получения и загрузки переменных сред разработки.

## Установка
Чтобы установить переменной среды разработки определена функция `Set`. Эта функция принимает ключ и значение для установки в системном среды разработки. Если значение установить невозможно, возвращается `KeyError`.

## Получение
Чтобы получить переменной среды разработки определена функция `Get`. Эта функция принимает ключ и возвращает переменную, если таковая имеется, хранящуюся по указанному ключу в системном среды разработки. Если ключ не хранится в системном среды разработки, функция вызывает панику с `IOError`.

## Загрузка
Чтобы загрузить среды разработки из файла определены функции `From` или `Load`. `From` принимает `Path` и возвращает коллекцию `Variable`. Если путь существует, файл загружается в среде разработки, в противном случае результат загружается из текущего системного окружения. `Load` просто загружает текущее системная среда разработки.
