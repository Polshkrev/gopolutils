# Таблица
Пакет `Таблица` определяет утилиты, помогающие в создании, доступе, подключении, вставке и удалении &mdash; а также в других операциях &mdash; таблиц базы данных.
Пакет определяет интерфейсы, перечисления и функции, чтобы сделать взаимодействие с таблицей базы данных *настолько* простым.
## Интерфейсы
В этом пакете определены следующие интерфейсы:
- Таблица: Интерфейс для определения каждого метода, связанного с взаимодействием с таблицей базы данных. Таблица реализует интерфейс [Размерный](/docs/ru-RU/collections/sized.md).

- Создатель: Для создания таблицы в базе данных с заданным именем. Используется для определения схемы базы данных.

- Сеттер: Для определения открытых методов-сеттеров для имени таблицы и подключения.

- Геттер: Для определения открытых методов-геттеров для имени таблицы и подключения.

- Вставщик: Для определения методов, связанных с вставкой данных в таблицу.

- Аксессор: Определяет методы, связанные с получением данных из таблицы.

- Remover: Определяет методы, связанные с удалением данных из таблицы.

- Дроппер: Определяет способ удаления таблицы.

- Клоузер: Определяет способ закрытия соединения базой данных.
## Ячейка
Другие определенные типы &mdash; перечисления, такие как тип `ячейка`. Этот тип должен быть расширен любой конкретной реализацией таблицы. Единственная ячейка, определенная по умолчанию, &mdash; `Id`, которая равна строковому значению `"id"`.
Чтобы расширить перечисления `Ячейка` можно использовать аналогичное определение, как описано:
```go
import "github.com/Polshkrev/gopolutils/table"

var (
username Field = "username"
password Field = "password"
// Получает строковое представление ячеек, разделенных запятыми, в качестве столбцов для базы данных.
fieldString string = GetFields(table.Id, username, password)
)
```
## Связывать
Чтобы связывать к базе данных определена функция `Connect`. Эта функция принимает [Адрес](/docs/ru-RU/fayl/path.md) и перечисление `Driver` и возвращает либо дескриптор базы данных на основе внутреннего параллельного сопоставления, либо исключение.
