# Таблица
Реализация простого сопоставления ключ-значение.

### Вставить
Чтобы вставить пары ключ-значение в таблицу есть &mdash; конечно же &mdash; метод `Insert`. Этот метод вставляет заданное значение, сопоставленное заданному ключу, в таблицу. Если ключ уже есть в таблицу, возвращается `KeyError` и таблица не изменяется.

### Удалить
Чтоб удалить из таблицы есть простой метод `Remove`. Этот метод принимает ключ, сопоставленный со значением, которое нужно удалить. Если таблица пуста или если заданный ключ не существует в таблице, то возвращается `KeyError`. Если таблица пуста, возвращается `ValueError`.

### Доступить
Чтобы доступить к значению в таблице доступен метод `At`. Этот метод принимает заданный ключ, в который сопоставляется значение. Этот метод либо возвращает пойнтер на сопоставляемое значение и пойнтер ошибки nil; либо нулевой указатель на данные и ошибку `ValueError`, если таблица пуста, либо `KeyError`, если ключ отсутствует в таблицу. Также доступен метод `Update`. Эта функция принимает ключ, которому сопоставлено обновляемое значение, и новое значение, присваиваемое данному ключу. Вышеупомянутое соглашение об исключениях действует и для этого метода. Методы `Keys` и `Values` доступны для доступа к соответствующим свойствам таблицы.

### Реализовать
Эта структура реализует интерфейсы `Mapping`, `Sized`, и `View`.
