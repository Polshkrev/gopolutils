# Конкурентное Скопление
Интерфейс, существующий для определения стандарта, по которому должна реализовываться конкурентная линейная структура данных, такая как список. Конкурентное скопление также должно реализовывать интерфейс [Sized](/docs/ru-RU/collections/sized.md), а также интерфейс [View](/docs/ru-RU/collections/view.md).
