## Содержание
1. [Использование](#использование)

### Использование
В настоящее время поддерживается только один алгоритм сжатия &mdash; `gzip`. Для сжатия данных удобно определена функция `Gzip`. Эта функция принимает на вход срез байтов и возвращает либо сжатый срез байтов, либо `IOError`. Для распаковки данных удобно определена функция `Gunzip`. Эта функция принимает на вход сжатый срез байтов и возвращает либо несжатый срез байтов, либо `IOError`.
